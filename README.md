# $\color{green}{\text{base64lib}}$
This __Python 3__ library speedily converts your base 10 numbers to a base between 37 and 64.\
I made it for anyone who thinks the int() function is limiting, as it only goes up to 36,\
due to the confusion it would make on what table to follow.\
Mine follows the Natural Lexicographical Base64 Alphabet.

The library's primary advantage lies in its ability to handle higher bases, which can be particularly useful in various applications such as data encoding, cryptography, and other computational tasks that require a broader range of numeral systems. By utilizing Base64lib, developers can seamlessly convert integers into a more extensive set of bases without the ambiguity that often accompanies traditional methods. This functionality not only enhances the versatility of number representation but also simplifies the process for those who require more than what standard libraries offer.
